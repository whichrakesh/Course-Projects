Question [1 mark]: What is a cross-compiler? What is the host platform? What is the target platform? You can use the Internet to learn these.
A cross compiler is a compiler capable of creating executable code for a platform other than the one on which the compiler is running. Host platform is the platform where the cross-compiler is run. The target platform is the platform for which the code is compiled.
